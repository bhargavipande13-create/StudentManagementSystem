# Student Management System

## Overview

The Student Management System is a web application built using Python, Flask, and SQLite. It allows users to manage student records by adding, viewing, editing, and deleting information through a simple web interface.

This project was developed to strengthen my understanding of web development, databases, and version control while preparing for cloud engineering and Azure-based applications.

---

## Features

- Add new students
- View all student records
- Edit student information
- Delete student records
- Responsive interface using Bootstrap
- SQLite database integration
- Flask backend

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/StudentManagementSystem.git
```

Move into the project folder:

```bash
cd StudentManagementSystem
```

Create a virtual environment:

```bash
python3 -m venv venv
```

Activate it:

macOS/Linux

```bash
source venv/bin/activate
```

Install dependencies:

```bash
pip install flask
```

Run the application:

```bash
python app.py
```

Open:

```
http://127.0.0.1:5000
```
